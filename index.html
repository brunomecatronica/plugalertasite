Por que o PlugAlerta?

Monitoramento em Tempo Real

Detecta quedas de energia instantaneamente através do sensor ZMPT101B, com comunicação via MQTT para dados em tempo real e HTTP para alertas críticos.

Notificações Push

Receba alertas no seu smartphone mesmo com o app fechado, usando Firebase Cloud Messaging.

Vinculação por QR Code

Conecte facilmente suas tomadas ao seu perfil através de QR codes únicos para cada dispositivo.

Arquitetura Escalável

Backend hospedado no Render com custos baixos. Sistema de cache, otimização de queries e índices de banco de dados para máxima eficiência.

Segurança e Confiabilidade

Autenticação Firebase, rate limiting, validações de entrada, sanitização de dados e sistema robusto para proteção total.

Histórico e Analytics

Acompanhe padrões de energia, estatísticas de falhas e relatórios detalhados no Firestore com dashboard interativo.
